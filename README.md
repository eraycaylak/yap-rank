# YAP! Rank & İlerleme

> YAP! Telegram botu için Rank ve İlerleme mini uygulaması.

## Özellikler

- 🎮 3D dönen rank rozeti
- ✨ Animasyonlu progress bar'lar (XP, Seri, Görev, Rozet)
- 🏆 Rank merdiveni (20 seviye)
- 🎊 Achievement popup bildirimleri
- 💫 Particle & floating SVG efektleri
- 📊 İstatistik kartları
- 🌌 Game UI (Orbitron + Rajdhani font)

## Netlify Deploy

Bu repo Netlify'a bağlı şekilde çalışır. Ana branch'a push = otomatik deploy.

## Geliştirme

Sadece `index.html` — framework yok, build step yok.

```bash
# Local önizleme
npx serve .
```
